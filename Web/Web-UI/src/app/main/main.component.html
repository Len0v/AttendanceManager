Attendance Manager

The technology wise attendance check system

About

Attendance Manager is an academic project that is being created at Poznań University of Technology. It focuses on creating centralized system for checking students attendance at various forms of classes using ELS - intelligent student card. It allows to register attendance using electronic card readers in which university computers are equipped. Web application allows you to view and modify all events that represent classes, as well as manually edit attendance lists if neccessary.

Tech
